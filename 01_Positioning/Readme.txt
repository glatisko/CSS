position:
static (default) - element placed based on the flow of page (cant be moved)
relative - moves left, rigth, top, bottom relative to default position (can use negative too), leaving empty space (does not move other elemetns)
absolute - moves absolutely within the containter that is not static, does not leave emtpy space (moves other elements)
         - resize element based on the text contained
         
fixed - moves absolutely within page
      - stays in position, regardless of scrolling

sticky - first positioned as normal, but once you scroll to specified location, stays there (eg. titlebar)
